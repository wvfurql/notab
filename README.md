## 它能帮你做什么

### 链接预览（核心功能）

按住快捷键点击链接，或者直接拖拽链接，当前页面就弹出小窗加载内容。看完关掉，不影响原来的页面，标签栏保持干净。

搜 Google 的时候特别好用——以前一个个点进去再返回，现在直接预览，5 秒判断有没有用，有用再正式打开，没用直接关掉，效率翻倍。

### 阅读模式

进入阅读模式，广告、侧边栏、推荐流全消失，只剩正文，干干净净。长文章读起来舒服很多。

### 视频模式

YouTube、B 站都能用。切换视频模式，画中画小窗出来，可以边看视频边继续浏览别的内容，互不干扰。

### 快速翻译 & 搜索

选中一段文字，直接拖动，小窗里立刻出翻译结果或搜索结果，不用切换页面。看英文资料的时候特别救命。

### 多链接同时预览

同时拖多个链接，多个小窗并排打开。买东西比价、查资料对比、读论文找引用，以前是 Arc 浏览器用户的专属体验，现在 Chrome 用户也能用上了。

### 窗口自由调整 & 固定

预览小窗可以拖动调整大小，也可以固定在屏幕某个角落，滚动页面它也不跑。
